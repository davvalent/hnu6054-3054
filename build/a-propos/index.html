Humanités numériques : Web sémantique et données. À propos de ce site.

Ce site web est une ressource pédagogique mise à la disposition des étudiantes et des étudiantes du cours « Humanités numériques : Web sémantique et données », donné au trimestre d’été 2024 par David Valentine à l’Université de Montréal.

Le contenu pédagogique de ce site est une adaptation du contenu de l’atelier « EDN 6103 — Web sémantique pour l’édition numérique » donné par Emmanuel Château-Dutier et Antoine Fauchié (CC BY-NC-SA 2018-2022).

Le contenu de l’atelier « HNU6054 — Humanités numériques : Web sémantique et données » est mis à disposition sous licence CC BY-NC-SA par David Valentine (2023-2024).

Le design de ce site web est inspiré du thème Lines par Ronalds Vilciņš.
